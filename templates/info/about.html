About Kolormodo.com

Kolormodo.com was created by Titus Stone. It was first launched live with a partial feature set during Late Summer 2010.

The site is written in Python on top of the Tornado web framework running on Google App Engine.  It was written in, as you'd expect, Komodo Edit. The themes Happy Day and Fancy Terminal were actually created so that I'd have something nice to stare at while coding the site (I was tired of my previous theme).

History

Komodo Edit by ActiveState has easily found itself to be one of my favorite editors for non-Microsoft languages. It's a great blend of necessary features without being overly complex or bloated (I'm looking at you Eclipse). One of the outstanding features that I really like is the ability to customize the color schemes.

However it didn't seem as if there was any stand-out place to find these themes, much less search and keep track of updates to ones I liked. The other problem is that searching google for themes often turned up more dead links than it did live ones and often live links didn't include color schemes for languages I wrote in.

Kolormodo.com was started to try and address these issues and to give Komodo users a singular place to find, share, and search themes.
